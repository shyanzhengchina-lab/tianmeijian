## 常见问题

### Q1: 如何开始编写测试用例？

**A**: 参考《测试用例编写指南》中的模板和示例，遵循AAA原则（Arrange、Act、Assert）。

### Q2: 如何测试共享组件？

**A**: 参考《组件测试指南》，书中提供了DataTable、FormModal、DetailDrawer等共享组件的完整测试示例。

### Q3: 如何进行集成测试？

**A**: 参考《集成测试场景设计》，书中提供了完整的业务流程测试场景和Cypress代码示例。

### Q4: 测试覆盖率要求是多少？

**A**: 根据《测试策略文档》，代码覆盖率要求≥80%，分支覆盖率≥75%。

### Q5: 如何准备测试数据？

**A**: 参考《测试用例编写指南》中的测试数据准备方法，包括工厂模式、构建器、数据库脚本等。
