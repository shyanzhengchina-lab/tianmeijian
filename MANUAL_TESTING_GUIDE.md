# MES系统手动测试指南

**测试目的:** 全面验证MES系统的功能完整性、数据一致性和用户体验

**测试执行者:** 系统测试人员

## 测试准备

### 1. 测试环境检查清单

- [ ] **前端环境**
  - [ ] Node.js 已安装（版本 >= 16.x）
  - [ ] 前端项目可以正常启动（npm start）
  - [ ] 浏览器可以使用（推荐Chrome或Edge）
  - [ ] 前端代码编译无错误

- [ ] **后端环境**
  - [ ] 后端服务可以正常启动
  - [ ] 数据库连接正常
  - [ ] API接口可以正常访问

- [ ] **测试数据**
  - [ ] 数据库已初始化
  - [ ] 基础数据已准备（物料、单位、车间、班组等）
  - [ ] 测试用户账号已创建
  - [ ] 测试数据量充足（每个模块至少10条记录）
